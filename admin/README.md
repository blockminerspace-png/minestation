# Painel Administrativo - Genesis Miner

Este painel permite controlar (ligar/desligar) o Backend e o Frontend do projeto Genesis Miner de forma centralizada.

## Como usar

1.  Abra um terminal nesta pasta (`admin`).
2.  Execute `npm start`.
3.  Acesse **http://localhost:4000** no seu navegador.

## Funcionalidades

-   **Backend**: Inicia o servidor backend (porta 5000).
-   **Frontend**: Inicia o servidor de desenvolvimento frontend (Vite, porta 5173).
-   **Logs**: Visualize os logs de ambos os serviços em tempo real na mesma tela.

## Requisitos

-   Node.js instalado.
-   As pastas `../backend` e `../frontend` devem existir e ter suas dependências instaladas (`npm install` em cada uma).
