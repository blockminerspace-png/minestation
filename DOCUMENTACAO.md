# 📓 Documentação MineStation 1.4.1

Este projeto agora conta com uma documentação estruturada no formato de **Obsidian Vault**.

## 🚀 Como acessar
1.  Abra o aplicativo **Obsidian**.
2.  Clique em "Open folder as vault" (Abrir pasta como cofre).
3.  Selecione a pasta `Obsidian_Docs` na raiz deste projeto.

## 📂 O que está documentado?
- **Estrutura de Pastas**: Onde encontrar cada parte do código.
- **Backend**: Detalhes do servidor Node.js e rotas de API.
- **Frontend**: Explicação sobre o ecossistema React/Vite.
- **Admin**: Funcionalidades do painel de controle.
- **Banco de Dados**: Visão geral do PostgreSQL e esquemas.
- **Scripts**: Utilitários de manutenção.
