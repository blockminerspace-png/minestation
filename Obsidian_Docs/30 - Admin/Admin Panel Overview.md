# 👤 Painel Admin

O diretório `admin/` contém um servidor dedicado para a interface de administração do MineStation.

## 🚀 Estrutura
- **`server.js`**: Um servidor Node.js (Express) focado em tarefas administrativas.
- **`node_modules/`**: Dependências específicas do painel admin.

## 🛠 Funcionalidades Principais
O painel admin permite ao proprietário:
1.  **Monitorar Usuários**: Visualizar saldos, endereços de carteira e atividades.
2.  **Gerenciar Economia**: Alterar taxas de saque, preços de itens e configurações de moedas.
3.  **Sistema de Backup**: Acionar e baixar backups do banco de dados PostgreSQL.
4.  **Configuração de Jogos**: Ajustar parâmetros de mini-games e eventos sazonais.
